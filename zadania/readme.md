# Grupy zadań

Każda nazwa pliku z zadaniami składa się z:
- dwóch liter oznaczających grupę zadań
- dwóch cyfr stanowiących numer 

- `wpxx` - wprowadzenie
- `ifxx` - instrukcje warunkowe if
- `frxx` - pętla for
- `whxx` - pętla while
- `sdxx` - struktury danych
- `fnxx` - funkcja
- `pkxx` - obsługa plików
- `wlxx` - wyrażenia listowe
